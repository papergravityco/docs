title: 'Notifications'
description: 'View and manage your system notifications'
---

The **Notifications** page shows system notifications for your account. From here you can review recent activity, filter by time period, and mark notifications as read.

## Viewing notifications

Notifications appear in the Notification Center, ordered from most recent. Each notification shows:

| Field         | Description                               |
| ------------- | ----------------------------------------- |
| **Icon**      | Indicates the notification type.          |
| **Title**     | A short summary of the event.             |
| **Message**   | Additional detail about the notification. |
| **Timestamp** | When the notification was sent.           |

Unread notifications are highlighted and display a **NEW** badge.

## Notification types

| Type        | Description                                         |
| ----------- | --------------------------------------------------- |
| **Info**    | General information about your account or activity. |
| **Success** | Confirmation that an action completed successfully. |
| **Warning** | Something may need your attention.                  |
| **Error**   | An action failed or requires follow-up.             |

## Filtering by time period

Use the time filter buttons to control which notifications are shown:

| Filter            | Notifications shown                  |
| ----------------- | ------------------------------------ |
| **Last 30 days**  | Notifications from the past 30 days  |
| **Last 3 months** | Notifications from the past 3 months |
| **Last 6 months** | Notifications from the past 6 months |

Changing the time filter resets the list to the default number of displayed notifications.

## Loading more notifications

If more notifications exist than are currently shown, a **Load More** button appears at the bottom of the list. Click it to load additional notifications in batches.

## Marking notifications as read

- To mark a single notification as read, click the checkmark button on the right side of the notification.
- To mark all unread notifications as read at once, click **Mark All as Read** in the top-right corner. This button only appears when you have unread notifications.
