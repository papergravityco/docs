## Available integrations

### Storage

| Integration          | Description                                |
| -------------------- | ------------------------------------------ |
| Amazon S3            | Sync assets to and from an S3 bucket       |
| Google Cloud Storage | Use GCS as a primary or backup asset store |
| Cloudflare R2        | Low-latency global asset delivery          |

### CMS & marketing

| Integration | Description                                     |
| ----------- | ----------------------------------------------- |
| WordPress   | Publish and manage images directly from Pixlayo |
| Webflow     | Sync assets to your Webflow CMS                 |
| HubSpot     | Insert approved assets into marketing campaigns |

## Setting up an integration

<Steps>
  <Step title='Open integrations'>
    In the Pixlayo dashboard, go to **Settings > Integrations**.
  </Step>
  <Step title='Choose a service'>
    Click the integration you want to enable and follow the authentication prompts.
  </Step>
  <Step title='Configure sync rules'>
    Define which folders or asset types should sync and at what frequency.
  </Step>
</Steps>
